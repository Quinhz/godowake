# GodoWake
An alarm clock made in Godot 3.6 with no experience.

# The Challenge
The goal is to build a functional and fun mobile alarm clock from scratch in 7 days, learning Godot and GDScript along the way.

## Planned Features
- [ ] Show Current time (v0.1.0-alpha)
- [ ] Load Custom Wallpaper (v0.1.0-alpha)
